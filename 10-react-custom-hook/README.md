# React 커스텀 훅

React 컴포넌트를 복잡하게 하는 코드 로직을 분리해 커스텀 훅으로 작성합니다.

- [ ] useStatusMessages 훅
- [ ] useImageType 훅
- [ ] useReactImage 훅
- [ ] useReactLibrary 훅

## 구현 사항

1. App 컴포넌트에서 상태를 관리하지 않도록 합니다.
1. DescriptionList 컴포넌트에서 커스텀 훅을 활용해 자체적으로 상태를 관리합니다.
1. ScrollButton, ScrollButton.Group 컴포넌트 자체적으로 "스크롤 무빙"을 관리합니다.
